# Experiment Log: Medical Domain Pre-training (PT-01)

## 1. 实验概述 (Overview)
本阶段为 **Qwen2.5-VL-7B-Instruct** 模型的初步垂直领域增强预训练（Domain-specific Pre-training）。实验的核心目标是通过灌输 15 本高质量医学教材知识，重塑模型的底层概率分布，使其从通用领域助手向医学垂直领域模型演进。

## 2. 数据集审计 (Dataset Audit)
* **数据集条目**: `medical_books_all`
* **语料来源**: 15 本临床医学核心教材（覆盖内科、外科、病理、儿科等核心学科）。
* **样本规模**: 
    * `max_samples`: 100,000 (采样上限)
    * `cutoff_len`: 2,048 (支持长上下文关联学习)
* **工程优化**:
    * **Packing**: 开启（`packing: true`），显著提升了训练过程中的 Token 吞吐率。
    * **Preprocessing**: 16 线程并发处理，降低数据加载瓶颈。

## 3. 训练参数配置 (Hyperparameters)

### 3.1 核心方法
| 参数 | 数值 | 说明 |
| :--- | :--- | :--- |
| **Stage** | `pt` | 预训练阶段 (Pre-training) |
| **Finetuning Type** | `lora` | 高秩 LoRA 微调 |
| **LoRA Rank** | `128` | **高秩配置**，旨在预训练阶段保留极高的表达空间 |
| **LoRA Target** | `all` | 覆盖所有线性层，确保全量知识注入 |

### 3.2 优化策略
* **学习率 (LR)**: `2e-5` (余弦退火 `cosine`)
* **训练轮数 (Epochs)**: `3.0`
* **等效全局 Batch Size**: 16 (4 cards/acc)
* **算子优化**: `FlashAttention-2` (fa2)

## 4. 基准评估审计 (Baseline Evaluation Audit)

在正式启动 PT 之前，对原始基座模型进行了一次严厉的医学知识对齐测试（使用 `原模型比较.py`），结果作为本次实验的对照组（Control Group）。
